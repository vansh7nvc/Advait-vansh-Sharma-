
## 📝 Assignment Categories

- **Regular Tasks**: Weekly or periodic assignments from the club
- **Projects**: Larger, multi-week projects
- **Practice Problems**: Coding challenges and exercises
- **Submissions**: Completed work ready for review

## 🚀 Getting Started

1. Browse the relevant folders for specific assignments
2. Check individual folder READMEs for detailed instructions
3. Review submitted code and solutions
4. Track progress on ongoing tasks

## 📌 How to Use This Repository

- Each assignment/task should be organized in its own folder
- Include a brief description or comments in the code
- Document any dependencies or setup instructions
- Update this README as new assignments are added

## ✅ Assignment Tracking

Keep track of assignments by:
- Creating branches for new work
- Using commit messages to describe progress
- Adding labels or milestones for organization
- Regularly pushing updates

## 📚 Resources

- Refer to individual assignment folders for specific requirements
- Check for any additional documentation in subdirectories
- Review completed work for reference and learning
